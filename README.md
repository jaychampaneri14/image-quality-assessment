# Image Quality Assessment

> No-reference image quality assessment using BRISQUE and NIQE

## Overview
This project is part of a comprehensive AI/ML portfolio covering computer vision python.

## Features
- Production-ready implementation
- Well-documented code with examples

## Installation

```bash
git clone https://github.com/jaychampaneri14/image-quality-assessment.git
cd image-quality-assessment
pip install -r requirements.txt
python main.py
```

python main.py
